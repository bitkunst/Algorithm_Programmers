### Comment

JavaScript에 내장되어 있는 sort() 메소드의 정렬 알고리즘이 어떻게 구현되어 있는지 아직 자세히 모르겠다,,
<br>
정렬 알고리즘을 직접 구현하여 적용하는게 좋지 않을까,, 라는 생각을 해본다.
<br>

```js
//* 오름차순 정렬
Array.sort((a, b) => a - b);

//* 내림차순 정렬
Array.sort((a, b) => b - a);
```
